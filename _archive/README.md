# Archive

Old documents, deprecated planning files, and previous session logs that are kept for historical reference.

## Contents

- **old-planning/** — Agent planning documents from early development
- **old-sessions/** — Detailed session logs from April (now consolidated in `../session.md`)

## Why Archive?

To keep the root directory clean and focused on active work, while preserving historical context if needed for debugging or understanding past decisions.

## How to Use

If you need to understand why a decision was made:
1. Check `../session.md` for recent work
2. Check `../CLAUDE.md` for current instructions
3. Only come here if you need the original planning context from early development

## When to Clean Up

Periodically review archived files and move truly obsolete ones to a separate `_old` folder or delete them entirely.
